# Installing the Lazarus Delis D&D 5e Workshop Content Module

## Installation

1. Download the **`lazarus-delis-dandd5e-workshop-content`** folder.

2. Locate your **Foundry VTT User Data** folder.

3. Open the `Data/modules` folder inside your Foundry User Data directory.

For most Windows users, the default path is:

```txt
C:\Users\YourName\AppData\Local\FoundryVTT\Data\modules
```

4. Copy or move the **`lazarus-delis-dandd5e-workshop-content`** folder into the `modules` folder.

Your folder structure should look like this:

```txt
FoundryVTT
└── Data
    └── modules
        └── lazarus-delis-dandd5e-workshop-content
```

Or, using the full Windows path:

```txt
C:\Users\YourName\AppData\Local\FoundryVTT\Data\modules\lazarus-delis-dandd5e-workshop-content
```

5. Launch (or restart) Foundry VTT.

6. Open your world and enable the module:

**Game Settings → Manage Modules → Lazarus Delis D&D 5e Workshop Content**

Open and Enjoy! (Feeds 3 Men!)
